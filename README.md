# JS01-console-variable
자바스크립트의 기본인 변수와 출력에 대해 알아보자

>* "Hello Javascript"를 출력한다.
>* 17을 출력한다.

>* 변수 age에 정수 17을 할당한다
>* 상수 Name에 문자열로 자신의 이름을 할당한다
>* 변수를 사용하여 다음과 같은 문자열을 출력한다
>* ex) 안녕 내 이름은 "신민석" 이고 나이는 17 이야'
